What I dislike

C++

C++ is a TERRIBLE language which has 10000 different ways to do something that should be easy, simple and elegant, while C has only one way. Everything is convoluted and complicated.

Python

Python like C++, the same ugly language. The scope even depends on indentation. Doing everything is so easy that you can’t feel the joy of coding.

Electron applications

It’s so heavy. It could have just run in the browser, but instead I have to install an entire extra browser just to run a single program. As the numbers of Electron apps increases, the proprietary browser on my computer also grows. Why is my storage space being used to install browsers? And it’s so heavy that it doesn’t run well on my ThinkPad X220.

Physical contact

The physical contact from an acquaintance always make me feel violated. We are not that close. If someone tries to get too familiar too soon, like putting an arm around me, it instantly kills any goodwill I might have for them.

Stereotype

About region: “people who grow up in Tianjin, China can all perform crosstalk”. About gender: “Men must be masculine and women must be feminine”.
